# Changelog

All notable changes to Skill-Tango will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-28

### Added
- AI-powered personalized course generation with NVIDIA models
- Multi-chapter curriculum with lessons and exercises
- Assessment system with AI-graded responses
- Multimodal content support (text, audio, visual)
- Stripe integration with subscription + credit top-ups
- XP, streak, and progress tracking
- i18n support with English translations
- Privacy Policy and Terms of Service pages
- Admin dashboard with AI logs, audit logs, and model tests
- Rate limiting and usage limits
- Unit tests for Toast, Skeleton, CustomSelect, and LessonPlayer
- GitHub Actions CI/CD workflows
- Cookie consent banner
